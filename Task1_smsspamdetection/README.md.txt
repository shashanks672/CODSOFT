Task 1 - SMS Spam Detection (CODSOFT Internship)

This project builds a machine learning model to classify SMS messages 
as Spam or Ham (Not Spam).

Steps:
1. Load the dataset (spam.csv)
2. Clean and preprocess text
3. Convert text to numbers (TF-IDF)
4. Train Logistic Regression model
5. Test accuracy and predictions

How to run:
- Open spam_sms_detection.ipynb in Jupyter Notebook
- Run all cells one by one
