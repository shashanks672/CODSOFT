# Credit Card Fraud Detection

This project detects fraudulent transactions using a highly imbalanced dataset.

## Steps
- Exploratory Data Analysis (EDA)
- Data preprocessing and feature scaling
- Model training using Logistic Regression
- Performance evaluated with ROC-AUC, Precision-Recall

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
